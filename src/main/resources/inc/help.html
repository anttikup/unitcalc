Help

Operators

Supported operators: +, −, · (or *), /, and ^ (exponentiation).

Space is also regarded as multiplication. It differs from · in that it binds more thightly than /. So, 5 m / 3 s means (5 · m) / (3 · s).

Function-like operators: sqrt (square root), cbrt (cube root).

Symbols

To use a unit you must define a symbol for it. Currently there are only SI units. Symbols for base units (m, s, ...) are defined by default, and symbols for other multiples must be defined separately. To define a symbol for a unit:

cm: centimetre

Multiple definitions:

cm: centimetre; mm: millimetre; km: kilometre
