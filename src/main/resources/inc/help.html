Help

Symbols

You can always use the full name of a unit by quoting it.

2 "foot" + 3 "millimetre"

To define a symbol for the unit, set the full name as the value of a variable.

cm = "centimetre"

Operators

Supported operators: +, −, · (or *), /, and ^ (exponentiation).

Space is also regarded as multiplication. It differs from · in that it binds more thightly than /. So, 5 m / 3 s means (5 · m) / (3 · s).

Function-like operators: sqrt (square root), cbrt (cube root).

You can use ? to get the result in given unit.

22 cm / 93 ms ? m/s

Use = to set variables.

length = 9 cm + 3 in

Special variable _ always holds the last result.

You can use semicolon (;) to separate expressions.

cm = "centimetre"; mm = "millimetre"; km = "kilometre"
